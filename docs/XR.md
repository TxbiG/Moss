# XR (Virtual/Augmented Reality)
## Overview
Moss XR is built on top of OpenXR SDK, the open standard for accessing virtual and augmented reality platforms and devices.
By leveraging OpenXR, Moss XR enables developers to write XR applications that are portable, maintainable, and device-agnostic.

> [!NOTE]  
> Ensure you have [OpenXR SDK](https://sourceforge.net/projects/openxr-sdk.mirror/) installed
