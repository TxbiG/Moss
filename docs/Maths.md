## Random Number Generations
Random number generation uses a pseudo-random number Generation
#### Random Integer
```cpp
int randi()
```

#### Random Float
```cpp
float randf()
```

#### Random Interger range
```cpp
int randi_range(int min, int max)
```

#### Random float range
```cpp
int randf_range(float min, float max)
```
